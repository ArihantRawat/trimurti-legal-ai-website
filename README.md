# Trimurti Marketing Website

A fully static Flutter web intro site for Trimurti and Juridical Loop OS.

The application contains no backend, authentication, databases, APIs, or network calls. Product surfaces are front-end concept previews rendered with Flutter widgets.

## Run

```powershell
C:\Users\ariha\flutter\bin\flutter.bat run -d chrome
```

## Build

```powershell
C:\Users\ariha\flutter\bin\flutter.bat build web
```
